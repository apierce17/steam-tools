Is this site safe?

Yes, I do not ask for users to sign in or submit any information that is not already public.
I could add in the ability for this in the future to prevent some errors such as friends list not showing but I have no intention to add this feature at this time until I am able to safely allow users to sign in.

How does this site have my Steam information?

I utilize the Steam API to pull in PUBLIC data, no private information is pulled in.
If you would like to see what information I am given to display or how their API works visit this link.

How do I get my Steam ID or Vanity ID?

There are a couple different ways:
Through the Steam website:
1. Go to your profile or search the profile here.
2. At the top grab the end of the user profile URL, it may be a name or a line of numbers.
https://Steamcommunity.com/id/GET-THIS

Through the Steam desktop app:
1. Hover over 'Steam' in the top nav..
2. Click on 'Settings'.
3. Click on 'Interface'.
4. Make sure 'Display web address bars when available' is checked
5. Close the settings window and find the desired profile.
6. At the top grab the end of the user profile URL, it may be a name or a line of numbers.
https://Steamcommunity.com/id/GET-THIS

Why are games not loading?

There are a few potential reasons:
1. Since I can only pull in public data if an account has their library private it will not show.
2. On occassion the Steam API will not return games even if the players library is public, try searching it again later.

Why is my profile not showing?

Please ensure you are entering the correct Vanity ID or Steam ID.

Why is my friends list not showing?

Since I can only pull in public data if an account has their friends private it will not show.
